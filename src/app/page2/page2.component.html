How to install Angular (On windows)

Step 1: Install NPM(Node Package Manager) from this link

Step 2: Install angular-cli using NPM with this command: npm install -g angular-cli

Step 3: Create a new project with the command: ng new PROJECT_NAME where PROJECT_NAME is the name of your project

Step 4: Use ng build to compile and build your project

Step 5: Use ng serve to run your project
