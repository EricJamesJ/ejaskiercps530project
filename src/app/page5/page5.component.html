Conclusion

Rate your experience with the frameworks that you chose

  • Angular - 8/10
  • ExpressJs - 10/10

Did they perform to your expectations?

  • Angular - Yes, I have quite a lot of experience with AngularJs and I knew that Angular would be very different from AngularJs but it was still impressive how nicely everything is organized with Angular
  • ExpressJs - I had no idea what I was getting myself into with ExpressJs as I've never used it before but it's simplicity made it very nice to work with

Were they difficult to install or configure?

  • Installation for both frameworks was straight forward, configuration on the other hand is another story. To get everything working together on Heroku took quite a bit of Googling and trail and error.

Was it easy to create the page(s) with it?

  • Yes, creating pages with Angular is very easy as it allows you to split everything up into components.

Was the learning curve steep compared with regular HTML/CSS/Javascript/PHP?

  • No, Angular and ExpressJs allow for inclredibly easy development compared to basic web design with no frameworks. The only difficult part was learning typescript and how to conenct services and components and other dependencies together in Angular

Would you have done things differently in retrospect?

  • No, learning typescript is necessary for Angular development so at the end of the day, I don't regret having to learn it

Did you regret your choice of frameworks?

  • No, Angular is one of the most popular frameworks currently in existence making it a very useful tool to have knowledge on
