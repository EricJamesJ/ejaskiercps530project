FrontEnd FrameWork: Angular

Popularity

According to this graph from StackOverFlow, Angular is the second most popular framework

Advantages

  • Component based architecture that allows the creation of UI with single parts known as components
  • Components are easily reusble
  • The nature of components allow for simple unit tests
  • Since Components are easily decoupled from one another, they allow for easier implementaion

Disadvantages

  • Angular is not AngularJs therfore, for developers migrating from AngularJs, they'll have to rewrite everything
  • A bit of a learning curve when it comes to fully understanding components, modules, dependency injection, etc...
  • Need to learn TypeScript

Which developers mostly use it and for what kind of websites

  • Web and Mobile apps
  • Large companies such as Google, Microsoft Office Home, Forbes, Blender Video, Xbox, Santander Bank, and BMW
